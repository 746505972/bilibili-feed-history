# Privacy Policy for Bilibili Feed History

> Last updated: 2026-01-27

## 1. Overview

Bilibili Feed History is a browser extension designed to help users save and view the history of recommended videos on the Bilibili homepage, allowing users to find videos that were previously shown but refreshed.

We respect your privacy and are committed to protecting it. This extension does not collect, transmit, or share any personal data.

## 2. Data Collection and Usage

This extension may access and process the following data locally on your device only:

Video titles, links, and related metadata visible on the Bilibili homepage

Extension settings and stored history records

All data is:

Stored locally using the browser’s storage API

Never transmitted to any external server

Never accessed by the developer

## 3. Data Sharing

We do not sell, transfer, or share any user data with third parties.
No analytics, tracking, advertising, or remote logging services are used.

## 4. Permissions Explanation

The extension requests the following permissions:

storage:
Used to store video history locally on the user’s device.

https://www.bilibili.com/
*:
Used to read and process content on the Bilibili website to detect and save recommended videos.

These permissions are used solely to provide the core functionality of the extension.

## 5. Data Security

All data remains on the user’s local device and is protected by the browser’s built-in security mechanisms.
No external data transmission occurs.

## 6. Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes.
Any updates will be reflected on this page.
