# Bilibili Feed History（B站推荐历史）

找回B站刷新掉的推荐视频

## 功能特性

你是否经常不小心点击了`换一换`按钮导致找不回首页想看的推荐视频？

Bilibili Feed History 浏览器扩展可以帮助你解决这个问题！它会自动保存你在B站首页看到的推荐视频，在你误操作刷新后仍能找回之前的推荐内容。

### 主要功能：

- **自动保存推荐视频**：自动捕获并保存出现在B站首页的推荐视频卡片
- **历史记录面板**：侧边栏面板展示所有保存过的推荐视频
- **时间标记**：每个视频都有保存时间戳，方便追溯
- **3列网格布局**：优化的界面布局，一次可查看更多视频
- **智能去重**：避免重复保存相同的视频内容
- **隐私安全**：所有数据仅保存在本地，不会上传到任何服务器

## 安装方法

1. 下载本扩展的所有文件
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择本项目文件夹

## 使用说明

- 扩展安装后会自动开始工作，保存你在B站首页看到的推荐视频
- 左侧边缘有一个隐藏的按钮，悬停时会展开历史记录面板
- 在历史记录面板中可以查看所有保存过的推荐视频
- 视频按时间倒序排列，最新的保存记录在最上面

## 技术特点
- 使用`MutationObserver`监听DOM变化，自动捕获推荐视频
- 支持多种B站页面结构，兼容不同的页面布局
- 自动清理克隆元素中的潜在安全风险
- 采用轻量级本地存储，不影响浏览器性能
